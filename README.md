# beyond-cpg-client-project-
This is a project for emerging brands into scale into real business website built using Webflow, featuring responsive design and smooth animations. Developed and customized for a client to deliver modern digital solutions.

This project is a modern digital service provider website developed using Webflow. It was created as a client-based project to showcase and deliver digital solutions in a clean, professional, and user-friendly way.

I was responsible for both the design and development, including customizing the layout and structure to meet the client’s requirements.

The website includes:

--> Fully responsive design for all devices --> Smooth and engaging animations --> Clean UI focused on user experience --> Structured layout for presenting digital services effectively

The main goal of this project was to build a visually appealing and functional platform that helps users easily find and explore digital solutions.
